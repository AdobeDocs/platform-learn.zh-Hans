# 架构和体验数据模型(XDM)概述

体验数据模型(XDM)是用于描述客户体验的标准数据模型。 当您使用XDM来构建模式并引入符合这些模式的数据时，您可以开始使用XDM系统。 XDM系统是支持Adobe Experience Platform中实时客户配置文件、机器学习和Customer Journey Analytics的基础架构。

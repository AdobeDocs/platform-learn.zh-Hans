# 在UI中创建并发布经过培训的模型

了解如何使用JupyterLab方法生成器笔记本制作的方法创建、培训、评估和发布模型。

在Adobe Experience Platform Data Science Workspace中，通过合并适合模型意图的现有方法来创建机器学习模型。 然后对模型进行训练和评估，以通过微调其相关的超参数来优化其运行效率和效能。 方法可重复使用，这意味着可以使用单个方法创建多个模型并针对特定目的进行量身定制。

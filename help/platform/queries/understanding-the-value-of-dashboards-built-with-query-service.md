# 了解通过查询服务构建的功能板的价值

借助Adobe Experience Platform，品牌厂商可以摄取、存储、构造和提取所有存储的数据集，包括行为、CRM、销售点数据等。 借助Experience Platform的查询服务，品牌商可以查询这些数据集并回答有关业务的特定问题，然后开始产生有影响力的见解。 此视频演示了使用查询服务在商业智能(BI)工具中构建功能板的价值。

# 合并模式概述

此视频概要介绍了Adobe Experience Platform实时客户资料使用的合并架构。 合并模式使您能够看到共享同一类的模式中包含的所有字段的合并。 Real-Time Customer Profile使用合并模式创建每个客户的整体视图。

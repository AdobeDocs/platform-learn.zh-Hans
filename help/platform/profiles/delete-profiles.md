# 删除配置文件

了解如何使用实时客户个人资料API从个人资料存储中删除数据。 通过使用配置文件API，您可以从配置文件存储中删除数据，而不会影响数据湖或身份图。 在排查身份图问题并更正只影响少数用户档案的数据摄取中的偶尔错误时，这可能很有帮助。

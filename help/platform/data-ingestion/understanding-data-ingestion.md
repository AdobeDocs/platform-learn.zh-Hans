# 数据引入概述

Adobe Experience Platform的数据摄取功能让您可以将数据整合到一个开放且可伸缩的平台中，以便管理实时客户档案。 有关详细信息，请访问[数据摄取文档](https://experienceleague.adobe.com/docs/experience-platform/ingestion/home.html?lang=zh-Hans)。

# 用例行动手册概述

了解用例行动手册如何帮助Real-time Customer Data Platform和Journey Optimizer的客户更快实现价值。 用例行动手册是涵盖多个行业和营销目标的几十个模板的目录。 行动手册可生成受众、历程和架构等技术资产，然后可对其进行自定义并发布到其他沙盒。

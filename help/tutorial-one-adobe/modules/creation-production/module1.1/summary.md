# Firefly服务的摘要和优势

Firefly Services是一组全面的创作AI和创作API，可自动化工作流。 通过将这些功能嵌入到任何生产或体验工作流中，组织可以扩展内容生成、编辑和汇编，在保持质量和控制的同时自动化内容的生产。

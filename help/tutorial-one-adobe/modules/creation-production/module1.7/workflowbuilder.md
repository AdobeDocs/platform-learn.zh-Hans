# 1.7 Firefly自定义工作流

Firefly自定义工作流的主要重点放在创建和配置工作流的创意技术人员和解决方案构建者上。

通过Firefly自定义工作流，您可以：

- 构建工作流：使用带有手动或AI帮助的可视化界面创建多步骤生产工作流。
- 运行工作流：利用在浏览器之外运行的强大运行时大规模执行工作流，包括批处理。
- 部署工作流：将工作流发布为API端点或编程UI，以便集成到其他工具和系统中。
- 集成工作流：将工作流连接到DAM、激活流和内容供应链等企业系统。

支持的主要用例包括以下内容：

- 社交媒体渠道调整，例如为每个社交媒体平台和类型创建大小变量。
- 资产准备，例如准备要用于其他表面（如打印或网络）的资产。
- 内容重定位，例如刷新和重复使用可视化图表以重新吸引感兴趣的受众。
- 电子商务，例如标准化产品图像和为不同受众创建特定于区段的变体。

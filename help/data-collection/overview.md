# 数据收集教程

数据收集是Adobe推出的新一代网站标记和移动SDK管理功能。 数据收集为客户提供了一种简单的方式来部署和管理用来加强相关客户体验的分析、营销和广告解决方案。 使用这些视频和教程帮助学习此关键技术。
